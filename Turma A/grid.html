O que é CSS Grid?

O CSS Grid Layout é um sistema de layout bidimensional que organiza elementos em linhas e colunas. Ele é ideal para criar páginas completas, grades de imagens, dashboards e estruturas complexas com controle total do espaço.

Como funciona

Para usar o Grid, aplicamos display: grid; no container e definimos o número e o tamanho das colunas e linhas com as propriedades grid-template-columns e grid-template-rows.

Propriedades principais

  • display: grid; — ativa o layout de grade.
  • grid-template-columns — define quantas colunas e seus tamanhos.
  • grid-template-rows — define quantas linhas e seus tamanhos.
  • gap — define o espaçamento entre os elementos.
  • justify-items e align-items — alinham os itens dentro das células.
  • grid-template-areas — permite nomear áreas da grade para layouts mais organizados.
