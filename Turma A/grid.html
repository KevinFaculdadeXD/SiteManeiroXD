O que é o Grid?

O CSS Grid é um sistema de layout bidimensional que organiza elementos em linhas e colunas. Ele permite criar estruturas de página complexas de forma simples e responsiva.

Propriedades principais

  • Display: grid; - Ativa o modelo de grid;
  • grid-template-columns: - define o número e tamanho das colunas;
  • grid-template-rows: - define o número e tamanho das linhas;
  • gap: - define o espaçamento entre os elementos;
  • grid-column / grid-row: - controla o posicionamento de cada item.
